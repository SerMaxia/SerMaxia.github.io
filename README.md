# SerMaxia.github.io

This is the source for my GitHub Pages portfolio.

## Structure
- `index.html`: entry point
- `assets/css/styles.css`: styles
- `assets/js/main.js`: small enhancements (theme toggle, current year)

## Local preview
Open `index.html` directly in a browser or use a simple static server.

## Deployment
Push to `main` on the `SerMaxia/SerMaxia.github.io` repository. GitHub Pages serves from root.
